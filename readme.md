# Talking API with Golang in Docker container

>in progress...

You can just chat to api and teach some words

##### Used:
- [x] golang-fiberr

## Running with docker:
Build:
```bash
docker build . -t talkyapi
```
Run:
```bash
docker run -p 3000:3000 -it talkyapi
```

and thats it, just send your requests
